exportDataToCsv(List<List> data) → Future<List>
Will export whatever list to the app's data file as csv format Future function use await The list must be list within list i.e. dataToExport = [ 'Name', 's', 'City', 'John', 30, 'New York', 'Alice', 25, 'Los Angeles', 'Bob', 35, 'Chicago', ]; exportDataToCsv(dataToExport) The function return bool, String
